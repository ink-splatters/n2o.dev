N2O.JS

ВСТУП

Модуль n2o.js надає протокольний цикл N2O для WebSocket каналів на стороні JavaScript. Використовується стандартний ECMA клас WebSocket. Розмір n2o.js — 2120 байт.

КОНФІГУРАЦІЯ

API

token()

Повертає токен сесії, закешований в localStorage.

N2O_start()

Ця функція створює сире WebSocket з'єднання з N2O WS і надсилає текстовий пакет <<"N2O,",Token/binary>>, який викликає ініціалізацію сторінки NITRO протоколом.
